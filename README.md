## Set up Angular locally

npx @angular/cli@17 new front-end --no-standalone

cd front-end

npm start

http://localhost:4200/

## Set up nest server locally

nest new nest-proxy

cd nest-proxy

npm run start

http://localhost:3000/

Enable Hot Reloading (Optional)

npm run start:dev

Test the API

Test in ThunderClient or POSTMAN
